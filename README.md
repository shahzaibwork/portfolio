# 🌐 Shahzaib Arshad | Portfolio

A modern, responsive portfolio website showcasing my software engineering experience, projects, certifications, and technical skills.

## 🔗 Live Website

👉 https://ranashahzaib-portfolio.vercel.app

---

## 🚀 Features

- Modern responsive UI
- Smooth animations with Framer Motion
- Glassmorphism design
- Dark theme
- Project showcase
- Experience timeline
- Technical skills section
- Certifications
- Contact form (EmailJS)
- Resume download
- SEO optimized
- Open Graph & Twitter meta tags

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS

### Libraries

- Framer Motion
- React Icons
- React Type Animation
- EmailJS

### Deployment

- Vercel

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/shahzaibwork/portfolio.git
```

Go to the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
